Types of Tags

In html there are two types of tags

  1. Empty Tags-Which doesn't have any closing tag.

    2. <br> , <hr>.

  2. Container Tags-Contains content inside it.

    3. <h1> Content </h1> , <p> Content </p>.

Attributes

Html elements can have attributes, it is specified in the start tag of an element.

<a href="URL"> </a>, <html lang="en"> </html>.

Here "href" and "lang" is an attribute.

Html Style

Html elements can be modified using style such as color,font,size etc

<body style="background-color:blue;"> Content </body>

<body style="text-align:center;"> Content </body>
